# Recursive-backtracing-maze-generation
Maze generation based on recursive backtracing algorithm.

# Usage
1. Create an object of the RecursiveBacktracking class.
   ```c++
   RecursiveBacktracking maze(width, height);
   ```

3. Generate a maze.
   ```c++
   maze.generateMaze();
   ```
